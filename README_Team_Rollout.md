# Citrix Test Automation Runner - Team Rollout

## Recommended Distribution
Distribute the runner as a versioned portable Windows ZIP:

`Citrix_Test_Automation_Runner_v1.0.1.zip`

Each tester should extract the ZIP locally to:

`%USERPROFILE%\Documents\CitrixTestAutomationRunner`

The app does not require Python to be installed on tester machines when packaged with PyInstaller.

## Build A Release
From the project root on the build machine:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements-build.txt
.\scripts\build_release.ps1
```

The script creates:

- `release\Citrix_Test_Automation_Runner_v<version>\`
- `release\Citrix_Test_Automation_Runner_v<version>.zip`

## What Is Included
The release contains:

- `CitrixTestAutomationRunner.exe`
- `_internal\` runtime files generated by PyInstaller
- `config\config.json`
- `test_cases\`
- `README_Quick_Start.md`
- `README_Team_Rollout.md`
- `version.txt`

The release intentionally excludes:

- `.venv`
- old evidence
- old logs
- personal desktop history

## Tester Data Location
By default, screenshots, logs, reports, and desktop-name history are written to each tester's profile:

`%USERPROFILE%\Documents\CitrixTestAutomationRunner`

This prevents multiple testers from writing evidence into the shared release folder.

## Update Strategy
- For wait-time/config changes, replace `config\config.json`.
- For testcase-only changes, replace `test_cases\`.
- For UI/core logic changes, ship a new ZIP version.
- Keep prior evidence folders untouched during upgrades.

## Security Validation
Before sending the ZIP to all testers:

1. Share it with IT/security for approval or hash whitelisting.
2. Pilot with two testers.
3. Confirm:
   - EXE launches without Python installed.
   - Citrix window activation works.
   - Screenshots are saved.
   - Clipboard copy works.
   - Word report is generated.
   - Endpoint security does not quarantine the app.

If unsigned EXEs are blocked, ask IT to deploy the same release folder through the approved software distribution channel.
